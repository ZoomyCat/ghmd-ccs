## How to get a copy of ghmd-ccs for your website.

### Common `git clone` methods:

#### For https clone:
```
git clone https://github.com/edge226/ghmd-ccs.git
```

#### For ssh https:
```
git clone git@github.com:edge226/ghmd-ccs.git
```

#### Checkout master:
```
git checkout master
```
